# Accomplish™ - ओपन सोर्स AI डेस्कटॉप एजेंट

Accomplish एक ओपन सोर्स AI डेस्कटॉप एजेंट है जो आपकी मशीन पर स्थानीय रूप से फ़ाइल प्रबंधन, दस्तावेज़ निर्माण और ब्राउज़र कार्यों को स्वचालित करता है। अपनी खुद की API कुंजियाँ लाएँ (OpenAI, Anthropic, Google, xAI) या Ollama के माध्यम से स्थानीय मॉडल चलाएँ।

## अक्सर पूछे जाने वाले प्रश्न

**क्या Accomplish स्थानीय रूप से चलता है?**
हाँ। Accomplish आपकी मशीन पर स्थानीय रूप से चलता है और आप नियंत्रित करते हैं कि यह किन फ़ोल्डरों को एक्सेस कर सकता है।

**क्या मुझे API कुंजी की ज़रूरत है?**
आप अपनी खुद की API कुंजियाँ (OpenAI, Anthropic, Google, xAI, आदि) उपयोग कर सकते हैं या Ollama के माध्यम से स्थानीय मॉडल चला सकते हैं।

**क्या Accomplish मुफ़्त है?**
हाँ। Accomplish ओपन सोर्स है और MIT लाइसेंस प्राप्त है।

**कौन से प्लेटफ़ॉर्म समर्थित हैं?**
macOS (Apple Silicon) और Windows 11 अभी उपलब्ध हैं।

## डेवलपमेंट

```bash
pnpm install
pnpm dev
```

बस इतना ही।

<details>
<summary><strong>पूर्वापेक्षाएँ</strong></summary>

- Node.js 20+
- pnpm 9+

</details>

<details>
<summary><strong>सभी कमांड</strong></summary>

| कमांड                                  | विवरण                              |
| -------------------------------------- | ---------------------------------- |
| `pnpm dev`                             | डेव मोड में डेस्कटॉप ऐप चलाएँ     |
| `pnpm dev:clean`                       | क्लीन स्टार्ट के साथ डेव मोड      |
| `pnpm build`                           | सभी वर्कस्पेस बिल्ड करें          |
| `pnpm build:desktop`                   | केवल डेस्कटॉप ऐप बिल्ड करें       |
| `pnpm lint`                            | TypeScript जाँच                    |
| `pnpm typecheck`                       | टाइप सत्यापन                       |
| `pnpm -F @accomplish/desktop test:e2e` | Playwright E2E टेस्ट               |

</details>

<details>
<summary><strong>एनवायरनमेंट वेरिएबल</strong></summary>

| वेरिएबल            | विवरण                                      |
| ------------------ | ------------------------------------------ |
| `CLEAN_START=1`    | ऐप शुरू होने पर सभी संग्रहीत डेटा साफ़ करें |
| `E2E_SKIP_AUTH=1`  | ऑनबोर्डिंग फ़्लो छोड़ें (टेस्टिंग के लिए)  |

</details>

<details>
<summary><strong>आर्किटेक्चर</strong></summary>

```
apps/
  desktop/        # Electron ऐप (main + preload + renderer)
packages/
  shared/         # साझा TypeScript टाइप
```

डेस्कटॉप ऐप Electron का उपयोग करता है जिसमें Vite के माध्यम से बंडल किया गया React UI है। मुख्य प्रक्रिया कार्यों को निष्पादित करने के लिए `node-pty` का उपयोग करके [OpenCode](https://github.com/sst/opencode) CLI को चलाती है। API कुंजियाँ OS कीचेन में सुरक्षित रूप से संग्रहीत होती हैं।

विस्तृत आर्किटेक्चर दस्तावेज़ के लिए [CLAUDE.md](CLAUDE.md) देखें।

</details>
